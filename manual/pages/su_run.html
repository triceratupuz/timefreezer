Setup and Running

Setup

Before doing something be sure to have already installed the following:

  • Python 2.7
  • WxPython 2.8
  • Csound 6.XX

No installation required, just copy all the files (preserving folders structure) and (all dependancies above must be already installed).

Running

Before lauching the program make sure that all dependancies are correctly installed.

Execute TPZ_TimeFreezer.py python file from command line (python TPZ_TimeFreezer.py) or by double clicking on it if your system allows.

General Rule

With this kind of controls is possible to:

  • Input the value with keyboard and confirm it with enter
  • Change the value with arrow keys on the side of the control
  • Use it as virtual slider by left mouse double click and hold, move the mouse to change value, release mouse left button to set value.
