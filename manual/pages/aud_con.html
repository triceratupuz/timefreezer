Audio configuration

General rule

Audio setup/settings must be configured by editing the file .csound6rc in the root directory, content of the file must be in a cSound format. For available options see the Canonical Csound Reference Manual - Command-line Flags

Some Hints

--omacro:SRATE=XXXXX -set sample rate (audio rate) to match your system

--omacro:KRATE=XXX -set control rate

-iadc -set audio input device, if nothing is specified uses system default device

-odac - set audio output device, if nothing is specified uses system default device

-b - set number of samples stored in i/o software buffer, if nothing is specified uses default csound value

-B - set number of samples stored in i/o hardware buffer, if nothing is specified uses default csound value
